# Перетворення типів

Перетворення типів — це створення нового значення іншого типу на основі наявного. Перетворення потрібно, коли операції вимагають певного типу, наприклад: числа — для обчислень, рядки — для формування тексту.

**Складання чисел та "склеювання" рядків**: оператор `+` для чисел виконує додавання, а для рядків — конкатенацію. Якщо переплутати типи, результат буде іншим, ніж очікується.

```python
a = "2"
b = "3"
print(a + b) # конкатенація рядків

x = 2
y = 3
print(x + y) # додавання чисел
```

Результат:

```text
23
5
```

У Python для перетворення типів використовують функції `int()`, `float()`, `str()` та інші:

* `int(value)` — перетворює до цілого числа. Для дійсних чисел — відкидає дробову частину; для рядків — вимагає щоб рядок містив тільки цифри, за потреби зі знаком.

* `float(value)` — перетворює до дійсного числа. Рядок має містити коректний запис дійсного або цілого.

* `str(value)`** — створює текстове представлення значення.

## Коли перетворення потрібне

Перетворення рядка з цифрами на число для обчислень:

```python
pi_str = "3.141592653589793"
pi = float(pi_str)
result = 2 * pi
print("2 * pi =", result)
```

Результат:

```text
2 * pi = 6.283185307179586
```

Формування повідомлення: перетворення числа на рядок:

```python
count = 5
message = "Кількість: " + str(count)
print(message)
```

Результат:

```text
Кількість: 5
```
