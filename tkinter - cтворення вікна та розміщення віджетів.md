# tkinter - створення вікна та розміщення віджетів

tkinter — це бібліотека Python для створення графічних інтерфейсів користувача (GUI). Замість того, щоб працювати з текстом у консолі, ми можемо створити вікно з кнопками, полями введення та іншими елементами, з якими користувач може взаємодіяти. Це робить програми більш зручними та зрозумілими для користувачів.

tkinter є частиною стандартної бібліотеки Python, тому не потрібно встановлювати його окремо.

## Робота з об'єктами в tkinter

Під час роботи з графічним інтерфейсом ми будемо створювати та використовувати складні об'єкти, такі як вікна та віджети (елементи інтерфейсу).

Створення нових об'єктів виглядає як виклик функцій. Наприклад, частина програми, яка створює об'єкт вікна виглядає так:

```python
window = Tk()
```

Для взаємодії з вже створеними об'єктами потрібно викликати **методи** об'єктів (дії, які виконуються з об'єктом). Виклик методу схожий на виклик функції, але перед ім'ям методу пишеться `.` та вказується об'єкт, метод якого буде викликатись.
Наприклад, для встановлення заголовку вікна (об'єкту що зберігається у змінній `window`), можна викликати метод `title` і передати йому рядок заголовку як аргумент:

```python
window.title("Моя програма")
```

## Створення вікна

Для початку роботи з tkinter потрібно імпортувати модуль цієї бібліотеки. Команда `from tkinter import *` імпортує все, що потрібно для початку роботи, після чого можна створити вікно програми за допомогою виклику `Tk()`.

```python
from tkinter import *

window = Tk()

window.mainloop()
```

Результат:

![Результат](tkinter%20-%20cтворення%20вікна%20та%20розміщення%20віджетів/1.png)

Після запуску цієї програми з'явиться порожнє вікно. Рядок `window.mainloop()` необхідний для всіх програм з графічним інтерфейсом — він запускає цикл обробки подій, який дозволяє вікну реагувати на дії користувача та відображатися на екрані.

⚠️ Без виклику `mainloop()` вікно не з'явиться або одразу закриється.

## Налаштування властивостей вікна

Після створення вікна можна налаштувати його властивості, такі як заголовок або розмір. Заголовок вікна відображається у верхній частині вікна, він встановлюється методом `title`. Розмір вікна встановлюється методом `geometry`, який приймає рядок у форматі `"ШИРИНАxВИСОТА"` (наприклад, "300x200" означає ширину 300 пікселів та висоту 200 пікселів).

```python
from tkinter import *

window = Tk()
window.title("Моя програма")
window.geometry("300x200")

window.mainloop()
```

Результат:

![Результат](tkinter%20-%20cтворення%20вікна%20та%20розміщення%20віджетів/2.png)

У цьому прикладі встановлюється заголовок "Моя програма" та розмір вікна 300 на 200 пікселів.

## Додавання віджетів

Віджети — це елементи інтерфейсу, такі як текст, кнопки або поля введення. Найпростіший віджет — це `Label` (мітка), який відображає текст у вікні. Властивості віджета можна встановлювати передаючи їх як аргументи під час створення віджета. Різні віджети мають різні властивості — наприклад, `Label` має властивість `text` для відображення тексту.

```python
from tkinter import *

window = Tk()

label = Label(text="Привіт, світе!")
label.pack()

window.mainloop()
```

Результат:

![Результат](tkinter%20-%20cтворення%20вікна%20та%20розміщення%20віджетів/3.png)

У цьому прикладі створюється мітка з текстом "Привіт, світе!". Властивість `text` передається як аргументи під час створення віджета. Метод `pack()` розміщує віджет у вікні.

⚠️ Без виклику `pack()` віджет не буде видимим.

## Зміна властивостей за допомогою методу `configure`

Властивості вікон та віджетів можна змінювати після їх створення за допомогою методу `configure`. Це корисно, коли потрібно змінити зовнішній вигляд елементів інтерфейсу під час роботи програми або коли властивості встановлюються окремо від створення.

Багато властивостей є спільними для різних віджетів та вікон. Наприклад, властивість `bg` (скорочення від "background") встановлює фоновий колір і працює як для вікон, так і для більшості віджетів:

```python
from tkinter import *

window = Tk()
window.geometry("200x50")
window.configure(bg="blue")

label = Label(text="Привіт!")
label.pack()

label.configure(bg="yellow")

window.mainloop()
```

Результат:

![Результат](tkinter%20-%20cтворення%20вікна%20та%20розміщення%20віджетів/4.png)

У цьому прикладі фон вікна встановлюється у блакитний колір за допомогою `window.configure(bg="blue")`, а мітка отримує жовтий фон використовуючи `label.configure(bg="yellow")`.
