# Типи значень - рядки, цілі та дійсні числа

Програми, написані на Python, часто працюють з різними видами даних (значень). Ці види називаються типами. Типи існують для того, щоб комп’ютер знав, як поводитися зі значенням: як його зберігати, які операції дозволені і як саме вони працюють, а також як друкувати значення.

## Рядок (`str`)
Рядок — це текст у лапках. Лапки можуть бути одинарні `'...'` або подвійні `"..."`.

Рядки зручно використовувати для імен або назв предметів, повідомлень або будь-якого іншого тексту.

```python
print("Привіт")
print('Python — це весело!')
```

Результат:
```
Привіт
Python — це весело!
```

З рядками працюють текстові операції, наприклад "склеювання" (конкатенація):

```python
print("Привіт, " + "світе!")
```

Результат:
```
Привіт, світе!
```

## Ціле число (`int`)
Цілі числа — це числа без дробової частини: `-2`, `-1`, `0`, `1`, `2`, ...

Цілі числа підходять для підрахунків кількості предметів, порядкових номерів, або інших цілих величин.

```python
print(10)
print(2 + 3)
```

Результат:
```
10
5
```

До цілих чисел можна застосовувати арифметичні операції: додавання, віднімання, множення, ділення та інше.

## Дійсне число (`float`)
Дійсне число — це число з дробовою частиною. У Python використовується крапка як роздільник дробової частини.

Дійсні числа зручно використовувати для вимірювань, відсотків та інших значень.

```python
print(3.14)
print(0.5 + 0.25)
```

Результат:
```
3.14
0.75
```

У цій таблиці наведено найпоширеніші типи даних, які використовуються у Python.
