# Шпаргалка - списки

| Код в Python | Пояснення | Приклади |
|---|---|---|
| `[a, b, c]` | список з елементами `a`, `b`, `c` | `["тісто", "соус", "сир"]`<br>`[1, 2, 3]` |
| `l[i]` | отримати елемент списку `l` з індексом `i` (індекси починаються з `0`, негативні індекси позначають елементи з кінця списку) | `pizza = ["тісто", "соус", "сир"]`<br>`pizza[0] # перший елемент ("тісто")`<br>`pizza[2] # третій елемент ("сир")`<br>`pizza[-1] # останній елемент ("сир")`<br>`pizza[-2] # передостанній елемент ("соус")` |
| `len(l)` | кількість елементів у списку `l` | `numbers = [1, 2, 3]`<br>`len(numbers) # результат: 3` |
| `l[i] = x` | замінити елемент списку `l` з індексом `i` на `x` | `pizza = ["тісто", "соус", "сир"]`<br>`pizza[2] = "моцарела"`<br>`# pizza: ["тісто", "соус", "моцарела"]` |
| `l.append(x)` | додати елемент `x` у кінець списку `l` | `pizza = ["тісто", "соус", "сир"]`<br>`pizza.append("орегано") # pizza: ["тісто", "соус", "сир", "орегано"]` |
| `l.insert(i, x)` | вставити елемент `x` у позицію `i` списку `l` | `pizza = ["тісто", "соус", "сир"]`<br>`pizza.insert(2, "салямі") # pizza: ["тісто", "соус", "салямі", "сир"]` |
| `del l[i]` | видалити елемент з індексом `i` зі списку `l` | `pizza = ["тісто", "соус", "сир"]`<br>`del pizza[-1] # pizza: ["тісто", "соус"]` |
| `l[start:end]` | елементи від `start` (включно) до `end` (невключно) | `numbers = [1, 2, 3]`<br>`numbers[1:3] # результат: [2, 3]` |
| `l[:end]` | елементи від початку до `end` (невключно) | `numbers = [1, 2, 3]`<br>`numbers[:2] # результат: [1, 2]` |
| `l[start:]` | елементи від `start` до кінця списку | `numbers = [1, 2, 3]`<br>`numbers[1:] # результат: [2, 3]` |

# Методи списків

| Метод | Пояснення | Приклади |
|---|---|---|
| `l.extend(x)` | додає всі елементи з іншої послідовності `x` в кінець списку | `numbers = [1, 2, 3]`<br>`numbers.extend([4, 5]) # numbers: [1, 2, 3, 4, 5]` |
| `l.remove(x)` | видаляє перше входження елемента `x` зі списку | `pizza = ["тісто", "соус", "сир"]`<br>`pizza.remove("соус") # pizza: ["тісто", "сир"]` |
| `l.pop(i)` | повертає та видаляє елемент з індексом `i` (якщо `i` не вказано, видаляє останній елемент) | `numbers = [1, 2, 3]`<br>`x = numbers.pop(1) # x: 2, numbers: [1, 3]` |
| `l.clear()` | видаляє всі елементи зі списку | `numbers = [1, 2, 3]`<br>`numbers.clear() # numbers: []` |
| `l.index(x)` | повертає індекс першого входження елемента `x` | `pizza = ["тісто", "соус", "сир"]`<br>`pos = pizza.index("соус") # результат: 1` |
| `l.count(x)` | повертає кількість входжень елемента `x` у списку | `numbers = [1, 2, 2, 3]`<br>`numbers.count(2) # результат: 2` |
| `l.sort()` | сортує список у порядку зростання | `numbers = [3, 1, 2]`<br>`numbers.sort() # numbers: [1, 2, 3]` |
| `l.reverse()` | змінює порядок елементів списку на протилежний | `numbers = [1, 2, 3]`<br>`numbers.reverse() # numbers: [3, 2, 1]` |
| `l.copy()` | повертає копію списку | `numbers = [1, 2, 3]`<br>`numbers_copy = numbers.copy() # numbers_copy: [1, 2, 3]` |
