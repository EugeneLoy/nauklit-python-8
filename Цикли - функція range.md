# Цикли - функція range

`range(n)` повертає послідовність від `0` до `n - 1`. Функція `range` може приймати два або три аргументи, щоб задати початок послідовності та крок. Це потрібно, коли треба використати послідовність чисел що починається не з нуля або потрібно використати числа з певним кроком (наприклад, лише парні).

## Послідовність від n до m

Виклик `range(n, m)` повертає послідовність цілих чисел від `n` (включно) до `m` не включно: `n`, `n + 1`, ..., `m - 1`.

Наприклад, надрукуємо числа від 2 до 5 включно:

```python
for i in range(2, 6):
    print(i)
```

Результат:

```text
2
3
4
5
```

## Крок у range

Виклик `range(n, m, k)` повертає числа від `n` (включно ) до `m` (не включно) з кроком `k`: кожне наступне число на `k` більше за попереднє. Якщо потрібні лише парні числа від 0 до якогось числа включно, зручно використати крок 2.

Наприклад, надрукуємо парні числа від 0 до 10 включно:

```python
for i in range(0, 11, 2):
    print(i)
```

Результат:

```text
0
2
4
6
8
10
```
