# Цикли for та списки

Цикл `for` можна використовувати не лише з `range`, а й зі списком. У такому випадку змінна циклу по черзі приймає значення кожного елемента списку. Це зручно, коли потрібно виконати одну й ту саму дію для всіх елементів списку.

## Перебір елементів списку

Щоб перебрати всі елементи списку, після `in` вказуйте ім'я списку. Змінна циклу на кожній ітерації отримує значення чергового елемента.

Наприклад, надрукуємо кожен фрукт із корзини:

```python
basket = ["яблуко", "груша", "банан", "слива", "персик"]

for fruit in basket:
    print(fruit)
```

Результат:

```text
яблуко
груша
банан
слива
персик
```

Змінна `fruit` спочатку дорівнює `"яблуко"`, потім `"груша"`, і так далі до останнього елемента.

## Перебір за індексами

Якщо в тілі циклу потрібен не лише елемент, а й його індекс у списку — можна використати `range(len(список))` для того, щоб отримати послідовність, що складається з усіх індексів списку. Функція `len(список)` повертає кількість елементів, тому `range(len(basket))` повертає індекси починаючи з `0` і до індексу останнього елемента. Елемент за індексом `i` отримують як `список[i]`.

Наприклад, надрукуємо кожен фрукт із вказанням порядкового номера:

```python
basket = ["яблуко", "груша", "банан", "слива", "персик"]

for i in range(len(basket)):
    print(i + 1, basket[i])
```

Результат:

```text
1 яблуко
2 груша
3 банан
4 слива
5 персик
```

Тут `i` — це індекс елемента (від 0 до 4), а порядковий номер для виведення отримано як `i + 1`. Через `basket[i]` отримуємо сам елемент списку за цим індексом.
