# Розгалуження - if та else

У програмах часто потрібно виконувати різні дії залежно від певної умови. В Python для цього використовується розгалуження - конструкція `if`. Розгалуження дозволяє перевірити умову (наприклад, результат порівняння) і обрати один з декількох шляхів виконання коду (гілок алгоритму).

## Просте розгалуження `if`

Конструкція `if` дозволяє виконати блок коду (гілку алгоритму) тільки тоді, коли умова істинна. Якщо умова хибна, цей блок просто пропускається.

```python
temperature = 33

if temperature > 30:
    print("На вулиці спека!")

print("Температура:", temperature)
```

Результат:

```text
На вулиці спека!
Температура: 33
```

Структура конструкції `if` складається з наступних частин:

* **Умова** - це вираз після ключового слова `if`, який обчислює логічне значення (`True` або `False`). У прикладі вище умова - це `temperature > 30`. Після умови обов'язково ставиться двокрапка `:`.
* **Блок `if` (гілка алгоритму)** - це один або декілька рядків коду, які виконуються, якщо умова істинна. Блок `if` пишеться з відступом (зазвичай 4 пробіли), після двокрапки. У прикладі вище блок `if` складається з одного рядка `print("На вулиці спека!")`.

⚠️ Усі рядки блоку повинні мати однаковий відступ. Коли відступ закінчується, закінчується і блок коду.

Якщо змінити `temperature` на значення не більше 30 - повідомлення "На вулиці спека!" не буде надруковано, тому що блок `if` пропускається.

Результат, якщо встановити `temperature = 10`:
```text
Температура: 10
```

## Розгалуження `if - else`

Коли в програмі потрібно обрати між двома взаємовиключними варіантами - до конструкції `if` можна додати `else`. Як і раніше, блок `if` виконається, тільки якщо умова істинна, а блок `else` - тільки якщо умова хибна.

```python
age = 16

if age >= 18:
    print("Людина доросла")
else:
    print("Людина неповнолітня")
```

Результат:

```text
Людина неповнолітня
```

У цьому прикладі спочатку перевіряється умова `age >= 18`. Якщо вона істинна (вік `18` або більше), виконується блок `if` і друкується "Людина доросла". Якщо умова хибна (вік менше `18`), виконується блок `else` і друкується "Людина неповнолітня".

Якщо змінити `age` на `20`, умова `age >= 18` стане істинною (`20` більше `18`), тому виконається блок `if` і буде надруковано "Людина доросла". Блок `else` у цьому випадку пропуститься.

Результат, якщо встановити `age = 20`:
```text
Людина доросла
```
