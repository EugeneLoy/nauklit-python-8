# Визначення функцій

Функції дозволяють об'єднати кілька команд в один блок коду, який можна використовувати багато разів. Замість того, щоб повторювати однаковий код у різних місцях програми, ми можемо визначити функцію один раз і викликати її скільки завгодно разів. Це робить код більш організованим, легшим для читання та зміни.

## Визначення простої функції

Функція визначається за допомогою ключового слова `def`, після якого йде назва функції та дужки. Назва функції повинна описувати, що вона робить. Після дужок ставиться двокрапка, а далі з відступом записується код, який буде виконуватися при виклику функції.

Наприклад, визначимо функцію, яка привітає користувача:

```python
def greet():
    print("Привіт!")
    print("Як справи?")
```

Нижче, після визначення функції, її можна викликати за назвою:

```python
greet()
```

Результат:
```
Привіт!
Як справи?
```

## Функції з аргументами

Функції можуть приймати значення, які називаються аргументами. Аргументи вказуються в дужках після назви функції і дозволяють передавати значення всередину функції. Завдяки аргументам одна й та сама функція може працювати з різними значеннями.

Наприклад, визначимо функцію, яка привітає користувача, використовуючи його ім'я:

```python
def greet(name):
    print("Привіт,", name + "!")
    print("Як справи?")
```

Після визначення функції, можемо викликати її передавши ім'я як аргумент:

```python
greet("Оля")
```

Результат:
```
Привіт, Оля!
Як справи?
```

У цьому прикладі значення `"Оля"` передається як аргумент `name`. Функція використовує це значення для друкування персоналізованого привітання.

Ту ж саму функцію можна викликати з різними значеннями імен, і вона буде працювати з кожним з них. Наприклад:

```python
greet("Максим")
greet("Анна")
```

Результат:
```
Привіт, Максим!
Як справи?
Привіт, Анна!
Як справи?
```

Кожен виклик функції `greet` використовує передане ім'я для створення персоналізованого привітання. Це демонструє, що один і той самий код може працювати з різними значеннями.

## Функції з кількома аргументами

Функції можуть приймати декілька аргументів. У такому випадку аргументи розділяють комами. При виклику функції значення передаються в тому ж порядку, в якому аргументи вказані при визначенні функції.

Наприклад, функція, яка друкує площу прямокутника:

```python
def print_area(width, height):
    area = width * height
    print("Площа прямокутника:", area)
```

При виклику функції ми передаємо значення для аргументів `width` та `height`:

```python
print_area(5, 3)
```

Результат:
```
Площа прямокутника: 15
```

У цьому прикладі значення `5` передається аргументу `width`, а значення `3` — аргументу `height`.

## Повернення значення за допомогою `return`

Функції можуть не тільки виконувати дії (наприклад, друкувати повідомлення), але й повертати результат обчислень. Для цього використовується ключове слово `return`. Коли функція досягає команди `return`, вона одразу завершує свою роботу і повертає вказане значення.

Наприклад, так буде виглядати функція, яка обчислює площу прямокутника і повертає результат:

```python
def calculate_area(width, height):
    area = width * height
    return area
```

Повернене значення можна використати далі в програмі:

```python
room_area = calculate_area(4, 5)
print("Площа кімнати:", room_area)
```

Результат:
```
Площа кімнати: 20
```

У цьому прикладі функція `calculate_area` обчислює площу прямокутника з шириною `4` та висотою `5` і повертає результат `20`. Це значення присвоюється змінній `room_area`, яку потім можна використати для друкування або подальших обчислень.
